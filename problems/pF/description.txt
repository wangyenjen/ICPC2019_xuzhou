# Adrien & Yen-Jen
---

## Problem Description

Adrien is a talented little child. 

He has a special hobby, that is, collecting numbered ping pong balls!

One day, Adrien is watching his treasures in his collection box. 
Inside the box are $N$ ping pong balls which numbered 1 to $N$ (each number is unique). 
Unfortunately, he found that one of the balls is disappeared.

At this time, he remembered that he saw Yen-Jen, who has a big round belly flash acrossing his home a few days ago. 

He realizes that Yen-Jen is the culprit!
Adrien, who loves the ball so much, how could he not have an argument with Yen-Jen?

However, there need to be some evidences before he argues with Yen-Jen. 

Therefore, he wants to know what is the number of the missing ball?


## Input Format
The first line contains an integer $N$ ($1 \le N \le 10^6$) - The initial number of ping pong balls in the box.
The second line contains $N$ distinct integers, $a_1, a_2, ..., a_N$ ($1 \le a_1, a_2, ..., a_N \le N$) -- the number of every ping pong balls in the box.

## Output Format

Print a integer - the number of the missing ball.

## Sample Input
```
3
3 1
```
## Sample Output
```
2
```
